# 📘 BELAJAR-VIBES-CODING

Repository ini dibuat sebagai dokumentasi pribadi untuk memahami dan menerapkan konsep Vibes Coding — sebuah pendekatan modern dalam pengembangan software dengan bantuan AI.

## 🎥 Sumber Belajar

Video referensi utama:
https://youtu.be/2hiaZghRYHk

## 🧠 Apa itu Vibes Coding?

Vibes Coding adalah pendekatan coding di mana kita:

- Mendeskripsikan apa yang kita inginkan, bukan bagaimana cara membuatnya
- Menggunakan AI sebagai eksekutor utama dalam penulisan kode
- Berfokus pada ide, logika, dan arah produk

## 🔄 Perubahan Mindset

Dari:

> "Saya harus menulis semua kode sendiri"

Menjadi:

> "Saya mengarahkan AI untuk membangun solusi"

## 👥 Peran Baru
- 🧭 **Programmer** → **Navigator** (mengarah & memberi instruksi)
- 🤖 **AI** → **Driver** (menulis & mengeksekusi kode)

## 🛠️ Setup Tools untuk Vibes Coding

Berikut tools yang digunakan:

- **GitHub** → untuk repository & kolaborasi
- **GitHub CLI** → untuk interaksi GitHub via terminal
- **Antigravity** → tools pendukung workflow AI (opsional sesuai kebutuhan)

## 📁 Setup Project

Penambahan cara setup awal:

1. Buat secara manual repositori di github 
2. Buat folder di local dan jalankan git init
3. Hubungkan folder di local dengan repo di github dengan cara git remote add origin <copy link .git nya>

## 🔁 Pola Kerja Vibes Coding

Pendekatan kerja dalam repo ini mengikuti pola berikut:

### 1. 📝 Planning dengan GitHub Issue

Sebelum menulis kode, selalu mulai dari perencanaan.

Gunakan GitHub Issue untuk:
- Ide fitur
- Rencana kerja
- Breakdown task
- Catatan eksplorasi

👉 **Tujuannya:** agar tidak coding tanpa arah

### 2. 🚫 Jangan Langsung Coding

Selalu lakukan ini dulu:
- Pahami masalah
- Tulis kebutuhan
- Buat rencana

Baru setelah itu:
➡️ Gunakan AI untuk implementasi

### 3. 🔄 Workflow Utama
1. Buat Issue (planning)
2. Diskusikan / refine ide (opsional)
3. Minta AI implementasi
4. Review hasil AI
5. Iterasi sampai sesuai
